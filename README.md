# **Hospitality Performance Dashboard | Power BI**

## Overview

The **Hospitality Performance Dashboard** is an interactive analytics solution built in **Power BI**, designed to analyze **revenue and operational performance** for hospitality properties across multiple **cities**, **room categories**, and **booking platforms**.

The dashboard provides a consolidated, decision-ready view of key hospitality KPIs to support **pricing, demand analysis, and operational decision-making**.
